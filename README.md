# Enhanced Pregnancy Chatbot

## Overview
This project is an enhanced chatbot designed to provide information and support for expectant mothers during their pregnancy. The chatbot utilizes advanced natural language processing techniques to understand and respond to user queries effectively.

## Features
- **24/7 Availability:** The chatbot is always available to answer questions.
- **Personalized Responses:** It can provide tailored information based on user input.
- **Comprehensive Resources:** Access to a wide range of resources and information about pregnancy.
- **Interactive Conversations:** Engages users in a conversational manner.

## Getting Started
### Prerequisites
- Python 3.x
- Required libraries (see `requirements.txt`)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/BTL5010TEJA/iproject.git
   ```
2. Navigate to the project directory:
   ```bash
   cd iproject
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
To run the chatbot:
```bash
python chatbot.py
```
